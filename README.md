# My-Excel-Project
📊 Excel Retail & Sales Analysis Project
A project completed during a Data Technician Bootcamp
📘 Overview
This project showcases core Excel skills developed during Week 1 of a Data Technician bootcamp. Using retail and sales datasets, I explored, cleaned, transformed, and analysed data using a range of essential Excel functions and tools. The work included hands‑on tasks across filtering, sorting, formulas, PivotTables, SWITCH logic, and visualisations.
The project strengthened my confidence in working with real‑world data and building insight‑driven outputs using spreadsheet tools.

🔧 Skills & Techniques Used
📐 Formulas & Functions
Used a range of common analytical formulas to calculate totals, averages, and extract structured information from the dataset:

SUM() — calculate commission totals
SUMIF() — conditional summation
AVERAGE() — average commission
AVERAGEIF() — conditional averages
Date functions:

DATE()
MONTH()
YEAR()


UNIQUE() — generate distinct lists from large datasets
VLOOKUP() — lookup and merge related information
SWITCH() — applied for categorising sales volume into High / Medium / Low


🧹 Data Cleaning & Preparation

Sorting age data from largest to smallest
Filtering datasets to focus on specific customer segments
Structuring consistent, analysis‑ready tables


📊 Data Analysis Using PivotTables
Created multiple PivotTables to summarise retail and bike‑sales data, including:

Sales by country, market, age group, and gender
Retail sales summarised by county and product
Insights into top‑performing regions and customer demographics

These PivotTables were used to answer analytical questions and identify key trends in the datasets.

📈 Data Visualisations
Developed clear and insightful charts to communicate performance metrics, including:

Bar and column charts
Line graphs tracking sales trends
Visual breakdowns of customer demographics
PivotCharts for dynamic filtering and exploration


🧠 Key Insights Identified
Throughout the tasks, I practised generating actionable insights—examples include:

Identifying product categories with the highest sales volume
Observing which markets certain countries performed strongest in
Comparing demographic differences in purchasing patterns
Spotting trends in high‑ vs low‑performing regions


📁 Project Contents
This project includes:

Excel workbook with completed tasks
PivotTables summarising product and regional performance
SWITCH‑based sales categorisation
Visualisations illustrating key findings
